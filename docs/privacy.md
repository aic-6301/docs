# プライバシーポリシー
あいしぃーが提供するサービス（以下,「本サービス」といいます。）における，ユーザーの個人情報の取扱いについて，以下のとおりプライバシーポリシー（以下，「本ポリシー」といいます。）を定めます。

## 第1条（個人情報）
「個人情報」とは，個人情報保護法にいう「個人情報」を指すものとし，第二条で記述されているものを指します。

## 第2条（収集する情報）
本サービスを利用するにあたり、以下の情報を取得することがあります。
- ユーザーID/ユーザー名
- メッセージログ (すべてのメッセージは保存されず、ログ機能の提供のみに使用されます。)
- 接続日時・接続元のIPアドレス (ゲームサービス)
- 不正行為防止のためのログ情報 (ゲームサービス)


## 第3条（個人情報を収集・利用する目的）
収集した際に使用する目的は以下の通りです。

- サービスの提供・運営のため
- ユーザーからの問い合わせ対応のため。
- 利用規約に違反したユーザーへの対処のため。
- トラブル発生時の対応のため。
- 荒らしや不正行為の調査・対応のため。
- 上記の利用目的に付随する目的

## 第4条（ログデータの保管期間）  
収集したログデータのうち、ゲームサービスにおけるログデータは、運営上必要な期間保存されます。
保存期間は定めず、管理者によって適宜削除されます。

## 第5条（個人情報の第三者提供）  
以下の場合を除き、第三者に提供することはありません。
- ユーザーの同意がある場合
- 法令に基づく開示要請。
- 利用目的の達成に必要な範囲で、個人情報の取り扱いを委託する場合
